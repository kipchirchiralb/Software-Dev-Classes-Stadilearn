Intro to CSS

CSS - Cascading style shee t - is a language used to enhance appearance of a web page(written in html)

How do we wrtie CSS (Language rules / Syntax)

      selector {
        property: value;
        property: value;
        property: value;
      }
  • selector - used to identify an element or a group of elements that you want to style
  • {} - a declaration block - used to group styles to be applied on the selected element
  • Property - the attribute you are interested in changing about an element e.g. background color, text size, spacing...
  • Value - the unit/actual description of a certain property

Where we put our CSS - Inline || Internal || External

  • Inline CSS - we write styles for individual elements using the style attribute
  • Internal CSS - we write styles in a style element(like below)- the styles can only apply to this(current) html page
  • External CSS(Recommended) - writing css in an a .css file that can be linked to many html pages. Allows max resuable code.
